# DirMove

Minimalistic lighweight app for moving files in selected directories to directories on the SD card. On the first run the app searches the filesystem and preselects directories on the internal storage that have a directory with the matching path on the SD card. The program keeps track of selected directories between runs. The "Move" button at the bottom triggers the move of the files from the selected directories to the directories with the same path on the SD card.

## Requirements
Android 8+
